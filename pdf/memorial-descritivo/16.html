JSON
JSON (JavaScript Object Notation - Notação de
Objetos JavaScript) é cada formatação simples de
troca de informações. Para seres humanos,é simples
de ler e escrever. Para máquinas, é fácil de entender
e gerar.Está baseado em um subconjunto da
linguagem de programação JavaScript, Standard
ECMA-262 3a Edição -Dezembro - 1999. JSON é
em formato texto e completamente independente de
linguagem, porque usa regras que são familiares às
linguagens C e familiares, incluindo C++, C#,Java,
JavaScript, Perl, Python e muitas outras. Estas
características fazem com que JSON seja um
modelo ideal de troca de dados.
Web services
Os World wide web services ficam com um sistema
operativo, plataforma de hardware ou linguagem de
programação de suporte Web.Apresentam apenas
informação, ou seja, jamais são aplicações Web que
suportam páginas que podem ser acendidas por
usuários através de navegadores Web.
Os Web services permitem reaproveitar sistemas já
existentes em uma organização e acrescentar-lhes
novas aplicações sem que seja necessário criar um
sistema a partir do zero. Assim, é possível aprimorar
os sistemas já existentes, incluindo mais informação e
novas funcionalidades de forma simples e rápida.
Incluindo em conta as operações disponíveis no Web
service, a aplicação solicita uma dessas ações. O
Web service efetua o processo e envia os dados
para a aplicação que requereu a operação.
A aplicação recebe os dados e faz a sua
interpretação, convertendo para a sua linguagem
própria.
Framework Spring
O Framework Spring é uma ferramenta capaz de
desenvolver a produtividade no desenvolvimento de
aplicações web, bem como contribuir para um
desenvolvimento que atenda aos principais padrões
de desenvolvimento.
Nos últimos anos, este framework vai se
consolidando cada vez mais e incluindo uma grande
quantidade de usuários desenvolvedores. Para
ilustrar o grande mercado alcançado por este
excelente framework, a REBELLABS realizou uma
pesquisa que mostra uma comparação de utilização
do Spring em relação a outros frameworks e o seu
grande desenvolvimento da comunidade.
O Spring Web Services (Spring-WS) é um produto da
comunidade Spring aplicado na criação de serviços
da Web orientados por documentos. O Spring Web
Services visa garantir o desenvolvimento de serviços
SOAP de primeiro acordo, permitindo a criação de
serviços da Web flexíveis usando uma das muitas
formas de manipular cargas úteis de XML. O produto
é baseado no próprio Spring, o que significa que
você pode usar os conceitos do Spring, como a
injeção de dependência, como parte integrante do
seu serviço da Web.
As pessoas usam o Spring-WS por diferentes razões,
mas a maioria é atraída por ele depois de obter
pilhas SOAP alternativas que faltam quando se trata
de seguir as maiores práticas de serviços da Web.
Firebase
De acordo com Viana (2018), o Firebase é uma
plataforma do Google que contém várias
ferramentas e uma excelente infraestrutura para
ajudar desenvolvedores web e mobile a criar
aplicações de alta qualidade e performance.
Esta plataforma contém quatro segmentos de
serviços: Analytics, Develop, Grow e Earn.
