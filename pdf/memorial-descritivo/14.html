React Native
React Native é, hoje, o mais parecido que uma
aplicação híbrida pode chegar de uma nativa. Pode
entregar nas duas plataformas dominantes (iOS e
Android) produtos de muita qualidade, com eficiência
e comportamento praticamente igual ao das
aplicações nativas voltadas para plataformas
Android e iOS.
Esse projeto ainda utiliza possibilidades modernas
para desenvolvimento front end, porém a arte ainda
em Java Script, uma linguagem de programação
muito conhecida e utilizada por ambos os sistemas
operacionais. No entanto, o stack o react é muito
poderoso uma vez que permite a aplicação de
diversas linguagens como o CSS Flexbox e JSX, além
de diferentes pacotes.
O react é indicado pelo Facebook, seu criador,
como uma biblioteca da linguagem de programação
Java Script, usada para a construção de interfaces.
No processo, ela faz exatamente isso,
proporcionando as ferramentas e padrões para que
os desenvolvedores possam criar aplicativos com
qualidade e de forma muito mais rápida.
HTML5
O termo “HTML5” é comum para a maior parte dos
usuários da Web de hoje em dia, e ainda que todos
nós tratamos com essa tecnologia quase que
cotidianamente, poucos sabem realmente dizer do
que se trata a mesma.
O HTML5 é a versão mais atual da linguagem de
programação conhecida como HTML (Hyper Text
Markup Language), sigla em Inglês para Linguagem
de Marcação de Hipertexto. Essa linguagem usa um
texto codificado para produzir e organizar conteúdo
online.
O HTML é considerado uma linguagem sólida e de
confiança pelos programadores e desenvolvedores,
e por isso também ficou o padrão de uso em todos os
navegadores atuais. Com o início do HTML5,
algumas melhorias foram introduzidas, mas foi
apenas em 2010 que esses novos recursos
começaram a ser utilizados com mais frequência.
Graças a essa popularidade, cada vez mais
diferentes plataformas, navegadores e sites estão se
tornando compatíveis com esse novo padrão.
O HTML é a linguagem de marcação que é usado
para estruturar e dar sentido para os conteúdos
web, definindo parágrafos, cabeçalhos, tabelas de
conteúdo, ou incluindo imagens e vídeos na página
web.
CCS3
O CCS3 é somente a versão mais atual do CSS, uma
evolução. Entre diversas as evoluções, ele incorpora
novos elementos para construir animações tanto em
2 como e 3 dimensões. Possui também novos recursos
para um maior controle sobre o uso com o qual se
apresenta mais características das páginas.
Com as avanços do CSS3, iniciou-se uma fase onde
o CSS é o maior sequaz dos Web Designers, pois
além dos aspectos básicos, é possível controlar
diversos outros aspectos no layout. Hoje, é possível
customizar muito além do básico, podemos
selecionar fundamentos específicos de um
determinado grupo de fundamentos; estruturar os
elementos independentemente da posição no
código HTML; influenciar na capacidade; colocar
gradiente em textos e formas; pôr sombras em textos
e outros elementos; inserir margens estilizadas;
regular a rotação e perspectiva; inserir animações.
Além disso, novas propriedades são inseridas
cotidianamente e sempre há como se atualizar e ir
usando novidades nas formatações do CSS em um
layout.

utilizada para aplicar estilo aos conteúdo HTML,
